# python-port-scanner-multithread
Fast Multi-threaded Port Scanner developed in Python for network auditing and security studies.

Python Multi-threaded Port Scanner
Ferramenta de reconhecimento de rede desenvolvida para identificar portas TCP abertas e mapear serviços ativos. Este projeto foca em performance através de multithreading e segurança na manipulação de conexões via sockets.

Funcionalidades
Alta Performance: Implementação de Threading e Queue para escaneamento simultâneo.

Resolução Dinâmica: Suporte a domínios (DNS) e endereços IP.

Mapeamento de Serviços: Identificação automática de protocolos comuns (HTTP, SSH, FTP, etc).

Tratamento de Erros: Gestão de timeouts e exceções de conexão para estabilidade.

Pré-requisitos
Python 3.6 ou superior instalado.
Não requer instalação de bibliotecas externas (Battery-Included).
